# RPI-PLC-Dashboard
Acquiring data from PLC in factory, store it in server SQL and visualise as dashboards on production line TV using django

<h1> Details </h1>
<h2> Hardware </h2>
<ul>
<li> Raspberry Pi 4B </li>
<li> TV </li>
<li> PLC Mitsubishi series Q with ETH </li>
</ul>
